# Product vision

Интеллектуальный ассистент — это не просто TODO-приложение, а помощник, который:

- подсказывает, что делать сейчас
- показывает риски по дедлайнам
- автоматически перераспределяет нагрузку

## Ключевые сущности

- Task
- Event
- Habit (в следующих итерациях)
- Workspace
- Project
- Prime task

## Основной пользовательский сценарий

1. Пользователь добавляет задачу или импортирует ее из внешнего источника.
2. Планировщик встраивает задачу в свободные окна.
3. Пользователь выполняет задачу через `play/stop/done`.
4. Система собирает статистику и дает рекомендации.

## Целевая аудитория v1

Студенты БГУИР с высокой учебной нагрузкой и большим количеством контрольных точек.
